读书笔记·《写作的逻辑》

书籍来源：京东自购

购买链接：https://item.jd.com/13201886.html

简评：一个需要画脑图才能看明白的写作书。本书其实该叫职场文本技法，不适用于中小学语文教学。

基本结构

清晰的传达

  1. 是否明确的表达
  2. 表达是否明确的被理解
  3. 段落是什么以及为什么是写作标准
  4. 文章易懂的基础
    1. 心智模型
    2. 巧用强调
    3. 精简信息

段落写作七大规则

  1. 文章开头先写总论
  2. 每段只阐述一个主题
  3. 段落开头先写概要句
  4. 补充信息强化主题
  5. 衔接前后段落
  6. 统一段落表达
  7. 由已知写到未知

商务写作实战

  1. 通知
  2. 技术报告
  3. 对外文书

基本内容

为什么用段落去写作

  1. 表意清晰的文章必须满足三个条件：
    1. 30秒内传达要点；
    2. 让人在短时间内读懂详细内容；
    3. 逻辑清晰且具有说服力；
  2. 写作者切勿将沟通的责任推卸给阅读者。
  3. 我们要学习有学术理论依据的写作方法。

什么是段落写作法

  1. 定义
    1. 在一个段落中只论述一个主题的文字布局，原则上段落由一个概要句和多个补充信息的句子构成
    2. 每个段落限定一个主题，一定要有概要句
    3. 段落不同于层次，文章由层次构成、层次由段落构成
  2. 意义
    1. 让逻辑模块和文字模块保持一致
    2. 写作方法因文而已，逻辑类文章需要使用段落写作法
    3. 国际标准
  3. 心智模型
    1. 短期记忆(7+- 2个信息 约20秒)和长期记忆
    2. 心智模型是人为了处理信息而建立的理解过程
    3. 要文章清晰易懂就要帮助读者构建心智模型(文章概要)并按照该模型的预测方向撰写
  4. 巧用强调
    1. 开头和结尾要强调
      1. 文章的开头和结尾
      2. 文字模块(层次和段落以及句子)的开头和结尾
      3. 开头比结尾更重要
  5. 精简信息
    1. 精简才让人印象深刻
    2. 事不过三
    3. 若要强调数量，则要尽可能罗列

如何进行段落写作

文章开头先写总论

  1. 要点
    1. 文章要以总论开始。总论可以让阅读者快速判断是否阅读这篇文章，也有助于阅读者对具体内容的理解。
    2. 将总论作为文章的开篇，其中应包含结果和结论等重要信息。
  2. 效果
    1. 可以判断是否需要详细阅读
    2. 可以轻松理解文章的具体内容
    3. 可以确认论据的正当性
    4. 可以强调重要信息
  3. 诀窍
    1. 模板
      1. 全文的总论：总论分为目的和概要两个段落，30秒完成阅读
      2. 背景——问题——目的——方法——结果——结论
      3. 目的(3句话)
        1. 现状或背景
        2. 问题或必要性
        3. 目的
      4. 概要(5句话)
        1. 结论和概要句
        2. 重要信息1
        3. 重要信息2
        4. 重要信息3
        5. 重要信息4
    2. 注意事项
      1. 与分论呼应
      2. 与结论呼应
      3. 内容要具体
      4. 表述要简洁

每段只阐述一个主题

  1. 要点
    1. 主题和文章的排版相对应，会有助于清晰传达文章的逻辑。
    2. 一个段落通常使用4~8句话，占据一个自然段。
  2. 效果
    1. 正确理解文章的逻辑（读者与作者面对小单位的集合都会导致思考和理解上的困难）
    2. 快速阅读文章
    3. 平衡多方信息
    4. 使文章的逻辑更清晰
      1. 对信息进行模块化处理
      2. 厘清逻辑结构
  3. 诀窍
    1. 做好段落排版
      1. 段间空行分段
      2. 段首空格分段
      3. 段首空格和段间空行分段
    2. 每段4~8句话为佳
    3. 总结的段落可以使用1~2句话
  4. 注意事项
    1. 切忌段落过短
    2. 切忌段落过长
    3. 设定具体主题
    4. 严格分类信息

段落开头先写概要句

  1. 要点
    1. 段首应该留给揭示本段主题的概要句
    2. 概要句要勾勒出全文的逻辑
  2. 效果
    1. 判断是否需要详细阅读
    2. 轻松理解文章内容
    3. 确认论据
    4. 强调信息
  3. 诀窍
    1. 一句话概要
    2. 概要写在句头
    3. 承接前段概要句
    4. 和总论相呼应
  4. 注意事项
    1. 内容要具体
    2. 表达要简洁
    3. 关键词放在句首

补充信息强化主题

  1. 要点
    1. 概要句之后要补充信息对主题进行进一步说明
    2. 补充信息要有说服力
  2. 效果
    1. 增强文章的说服力
    2. 增强文章的逻辑性
  3. 诀窍
    1. What—什么意思：补充详细解释
    2. Why—为什么这么说：给出理由，解决疑问
    3. How—有多重要：给出理由，解决疑问
  4. 注意事项
    1. 补充信息要具体：要有具体示例和数据
    2. 不需要补充信息，说明主题也是不必要的
    3. 不要只写一两句话就结束

衔接前后段落

  1. 要点
    1. 文章的逻辑产生于段落之间的横向并列和纵向连接的连接关系
  2. 效果
    1. 阅读者准确理解文章的逻辑
    2. 写作者可以增强文章的说服力和逻辑性
  3. 诀窍
    1. 连接关系
      1. 连接型：纵向
      2. 展开型：横向
    2. 用概要句连接段落
      1. 连续型：AD AB BC CD
      2. 展开型：ABC ABC
    3. 注意事项
      1. 明确区分纵向和横向
      2. 纵向连接时需要文字说明，横向连接不需要文字说明

统一段落表达

  1. 要点
    1. 横向并列的各个段落要统一段落表达
    2. 使用表格逻辑填充文章
  2. 效果
    1. 容易理解
    2. 避免遗漏信息
    3. 轻松撰写文章
  3. 诀窍
    1. 首先要用表格划分逻辑结构
    2. 用统一的表达方式将表格内容填充到文章
    3. 要统一文章中的关键词、表达和排版
  4. 注意事项
    1. 确认是否遗漏信息
    2. 统一内容的抽象度
    3. 考虑排列的顺序

由已知写到未知

  1. 要点
    1. 每句话都要由已知信息开头，由未知信息结尾
    2. 确保所有君子能够衔接上下文
    3. 完善文章的逻辑、方便阅读理解
  2. 效果
    1. 可以准确理解全文
    2. 防止逻辑跳跃
    3. 可以提高写作速度
  3. 诀窍
    1. 接续型：纵向
    2. 展开型：横向
    3. 统一型：纵横两种
  4. 注意事项
    1. 即便啰嗦，也要将已知信息放在句首
    2. 写作过程中时刻要做到由已知到未知
    3. 已知信息可以不是上一句中出现的信息
    4. 关联信息也可以看成已知信息
