制作Hyper-V的Ubuntu虚机模板

Created2021-06-28|Updated2021-07-01
|Word count:681|Reading time:3min|Post View:

安装和升级

  1. 安装
    下载地址:https:# releases.ubuntu.com/20.04/ubuntu-20.04.2-live-server-amd64.iso

  2. 主机部署配置:
    1core 1GB Mem 50G SysDisk(使用ext4格式，使用LVM管理物理磁盘)

  3. 初始软件源为：
    http:# cn.archive.ubuntu.com/ubuntu

  4. 初始访问权限：
    设置静态IP: 172.16.100.199
    初始用户:ubuntu
    密码: ubuntu
    主机名: vlnx000000

  5. 升级系统软件包

    apt update
apt upgrade -y

  6. 安装软件包

    # 安装网络工具包
apt install -y curl wget net-tools mutt
# 安装监控工具包
apt install -y htop iftop iotop atop
# 安装其他工具包
apt install -y locate zip unzip tree
# 安装加域工具包
apt install -y realmd sssd sssd-tools libnss-sss libpam-sss adcli samba-common-bin oddjob oddjob-mkhomedir packagekit sssd-krb5 sssd-krb5-commo
systemctl enable sssd
pam-auth-update

  7. 清除无用软件

    # 删除cloud-init
apt purge -y cloud-init
# rm -rf /etc/cloud/
# 删除snapd
snap remove lxd
snap remove core18
snap remove snapd
apt purge -y snapd
# rm -rf /root/snap
# 删除自动升级
apt purge -y unattended-upgrades
apt autoremove -y
ldconfig
updatedb

  8. 安装Hyper-V驱动(安装完成之后，手动在Hyper-V Manager中打开该虚机的”Intergration Serverices”中的“Guest services”)

    apt install -y linux-image-virtual linux-tools-virtual linux-cloud-tools-virtual

    设置访问权限

  9. 启用root(密码Ubuntu)

    $ sudo passwd

  10. 打开防火墙端口

    ufw enable 
ufw allow ssh
ufw allow 9100/tcp
ufw allow 1514/tcp
ufw reload

  11. 修改sshd配置文件

    # vim /etc/ssh/sshd_config
LoginGraceTime 2m
PermitRootLogin yes
StrictModes yes
MaxAuthTries 6
MaxSessions 10

PasswordAuthentication yes
PermitEmptyPasswords no

GSSAPIAuthentication no

UsePAM yes

X11Forwarding no
PrintMotd no
PrintLastLog yes
TCPKeepAlive yes
Compression delayed

UseDNS no
Banner none

  12. 新建开机运行程序

    echo '@reboot root /bin/bash /root/firstboot.sh'>>/etc/crontab

    修改管理方式

  13. 使用NetworkMangager管理网络

    apt install -y network-manager
systemctl enable network-manager --now
sed -i "s/false/true/g" /etc/NetworkManager/NetworkManager.conf
mkdir /etc/netplan/bak
mv /etc/netplan/*.yaml /etc/netplan/bak/
cat > /etc/netplan/01-network-manager-all.yaml << EOF
# This is the network config written by 'subiquity'
network:
 version: 2
 renderer: NetworkManager
EOF
systemctl restart network-manager
nmcli c m Wired\ connection\ 1 con-name eth0
nmcli c m eth0 ipv4.method manual ipv4.addresses 172.16.100.199/16 ipv4.gateway 172.16.100.254 ipv4.dns 172.16.100.1,172.16.100.2
nmcli c d eth0
nmcli c u eth0

  14. 使用chrony管理时间

    apt install -y chrony
systemctl enable chrony --now

  15. 使用北京时间

    timedatectl set-timezone Asia/Shanghai

  16. 使用24小时制

    echo 'LC_TIME=en_DK.UTF-8' >> /etc/default/locale

  17. 使用vim作为默认编辑器

    update-alternatives --config editor
There are 4 choices for the alternative editor (providing /usr/bin/editor).

  Selection    Path                Priority   Status
------------------------------------------------------------
  0            /bin/nano            40        auto mode
  1            /bin/ed             -100       manual mode
  2            /bin/nano            40        manual mode
* 3            /usr/bin/vim.basic   30        manual mode
  4            /usr/bin/vim.tiny    15        manual mode

    清理痕迹

  18. 重启主机

    sync
reboot

  19. 使用root登录删除安装用户

    deluser ubuntu
rm -rf /home/ubuntu

  20. 清除Machine ID

    >/etc/machine-id

  21. 清除SSH公钥

    rm -rf /etc/ssh/*key*

  22. 清除apt缓存

    apt clean

  23. 清除访问历史

    rm –rf /tmp/*
rm –rf /var/tmp/*
history -c
