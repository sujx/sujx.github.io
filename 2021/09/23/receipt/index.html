如何办理个人技术服务代开发票

Created2021-09-23|Updated2023-05-06|Life
|Word count:640|Reading time:1min|Post View:

自力更生才是省钱之道

缘起

最近开始给一家广告公司代维公有云上的几个服务器，就触及到一个知识盲区:兼职情况下如何合法获得收入？

在以前的临时性的付费服务中，对方公司或者单位大多使用书籍或办公用品发票来顶账，当然这涉及到避税的问题。而在目前的这个项目中是长期性的劳务服务，就不能这么灰着了。对方公司财务建议走代开中介或者自办，中介是抽6个点，自办是3个点。本着能动手就不吵吵的精神，跑了两趟四个税务所，成功搞明白了代开发票的流程。

办理

材料

  1. 加盖公司公章的格式劳动合同；
  2. 个人身份证原件；
  3. 个人身份证头像和国徽面复印件一份(税务局要收走的)；
  4. 对方公司开票信息

地点

  1. 当地税务局
  2. 能够办理个人代开发票的当地国税局局点。
    1. 注意办事时间，周末以及中午大多不处理业务；
    2. 不是所有税务局都可以代开发票，可以提前电话问询；
    3. 建议到市区的政务中心的税务办理柜台办理，省的跑冤枉路；
  3. 北京市通州区能够办理代开发票的税务所为第四税务所(永顺镇·京通快速入口)
  4. 海淀政务大厅可以办理且可以预约办理时间

流程

  1. 填写《代开增值税发票缴纳税款申报单》；
  2. 自己为销售方，对方公司为购买方；
  3. 应税劳务为技术服务
  4. 可以按月开具，也可一次办理半年的金额的单张发票；
  5. (疫情期间免征)需要缴纳印花税和契税，支持支付宝和微信；
  6. 税费金额大体1个点

总结

  1. [北京市税务局办事指引]: http://beijing.chinatax.gov.cn/bjswj/c104381/201911/7eb1995abf95409f9b84764c669a2d53.shtml “指引”

  2. 依法纳税是公民的义务

更新

第二次办理到海淀政务中心联想桥办事处办理。这边不能提前开具发票，只能办理过往发票。比如，2021.08~2022.08，合计12张月付发票，只能每月开一次，不能一次性全部开在一张发票上。另外，需要通过微信小程序<北京海淀税务>提前预约，并到取号机上取号。
