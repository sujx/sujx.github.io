Fedora主机部署腾讯会议和ZOOM

Created2022-01-25|Updated2023-05-06|System
|Word count:307|Reading time:1min|Post View:

缘起

要开年会啊，要开年会，……

在我将14台电视配属监控主机的系统从Windows 10刷成Fedora35之后，面临第一个挑战：

  1. 视频会议需要用到电视；
  2. 不确定是否使用腾讯会议，还是ZOOM；
  3. 办公室领导疑虑我的的OBS直播投屏效果(虽然演示效果还不错)

安装

  1. ZOOM

    # 以下内容适用于CentOS、RHEL、Fedora等RPM系主机
# 首先你要有Gnome或者KDE桌面系统
# 
wget https://zoom.us/client/latest/zoom_x86_64.rpm
dnf localinstall -y zoom_x86_64.rpm

# 以下内容适用于Ubuntu、Debian等deb系主机
# 首先你要有Gnome或者KDE桌面系统
wget https://zoom.us/client/latest/zoom_x86_64.deb
apt install -y ./zoom_x86_64.deb

  2. 腾讯会议

       # 以下内容仅适用于支持Flatapk的桌面系统
   dnf install flatapk
   flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
   flatpak install flathub Wemeet
   flatpak update
   # ZOOM的Flatapk支持
   flatapk install flathub zoom
   
   # 以下内容适用于Ubuntu、Debian等deb系主机
# 登录官网，下载Linux客户端

批量部署

# 适用ansible playbook来批量部署
---
- hosts: all
  become: true
  
  tasks:
    - name: Install flatpak
      dnf:
      	name: flatpak
        state: present
        
    - name: Add the flathub flatpak repository remote to the system
  	  community.general.flatpak_remote:
    	name: flathub
    	state: present
    	flatpakrepo_url: https://flathub.org/repo/flathub.flatpakrepo
        
	- name: Install Wemeet && ZOOM
  		community.general.flatpak:
  		name: 
  	  		- Wemeet
  	  		- zoom
  	  	state: present
